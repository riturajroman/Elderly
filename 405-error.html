405 - Method Not Allowed

The server is rejecting the POST request to the PHP file.

Common causes and solutions:
  • Server doesn't support PHP: Make sure you're on a PHP-enabled hosting service
  • File permissions: Ensure process-form.php has proper read/execute permissions (644 or 755)
  • Server configuration: Some servers block POST requests to certain files
  • ModSecurity rules: Security modules might be blocking the request

Troubleshooting steps:

  1. Test server compatibility:
    Visit: https://theelderlywellness.com/server-test.php
  2. Check file permissions:
    chmod 644 process-form.php
  3. Contact your hosting provider if the issue persists
  4. Alternative: Try renaming process-form.php to form-handler.php

For developers:

Check your hosting control panel for:

  • Error logs
  • PHP version and modules
  • ModSecurity settings
  • Server configuration
